Text field on light surfaces — label, helper/error text, optional leading/trailing adornments, green focus ring.

```jsx
<Input label="メールアドレス" placeholder="you@example.com" />
<Input label="クーポン" error="このコードは無効です" />
```

Forwards all native input props. Sibling controls: `Select`, `Checkbox`, `Radio`, `Switch`.
