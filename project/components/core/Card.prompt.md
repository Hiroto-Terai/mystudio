Generic light surface with the signature stacked-shadow halo. Use for panels, feature blocks, and tiers.

```jsx
<Card>標準カード</Card>
<Card variant="featured">おすすめプラン</Card>
<Card variant="band" elevation="flat">機能カテゴリ</Card>
```

`variant`: default / featured (aloe) / band (pistachio) / wash / hairline. `elevation`: flat / hairline / raised / halo. `radius`, `pad`, `as`.
