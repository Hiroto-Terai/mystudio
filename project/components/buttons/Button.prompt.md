Pill CTA button — the system's dominant action. Use for any primary/secondary action; shape is always a pill.

```jsx
<Button variant="primary" size="lg" onClick={checkout}>レジに進む</Button>
<Button variant="accent">カートに入れる</Button>
<Button variant="outline">ログイン</Button>
```

Variants: `primary` (ink pill), `accent` (green pill), `featured` (aloe pill), `outline` (hairline), `ghost` (text). Sizes: `sm`/`md`/`lg`. Props: `fullWidth`, `disabled`, `iconLeft`, `iconRight`, `as`.
